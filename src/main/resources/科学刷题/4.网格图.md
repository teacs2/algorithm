## 一、网格图 DFS

适用于需要计算连通块个数、大小的题目。

部分题目做法不止一种，也可以用 BFS 或并查集解决。

二叉树递归和网格图递归的区别：

|              | **二叉树**           | **网格图**               |
| ------------ | -------------------- | ------------------------ |
| **递归入口** | 根节点               | 网格图的某个格子         |
| **递归方向** | 左儿子和右儿子       | 一般为左右上下的相邻格子 |
| **递归边界** | 空节点（或者叶节点） | 出界、遇到障碍或者已访问 |
